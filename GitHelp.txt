команды только для одного проекта

git init        |   инициализация Гита
git remote add origin link_to_repository    |ДОБАЛЕНИЕ РЕПОЗИТОРИЯ
    прим ( git remote add origin https://github.com/demiurge41/OOP-60-2.git   )

git status  | статус репозитория

git add .   |   добавление файлов в буфер обмена
git commit -m "message" |сохранение всего из буфера обмена

git push origin master  | отправка всех сохранений в репозитории (на сайт GitHub)
